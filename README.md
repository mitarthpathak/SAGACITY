# SAGACITY 🌐🤖

Sagacity is a multilingual AI chatbot designed to improve **health awareness and accessibility** in rural and semi-urban communities.  
Built with scalable cloud infrastructure and conversational AI frameworks, it empowers users with reliable health information, symptom guidance, and awareness campaigns.

---

## ✨ Features
- **Multilingual Support**: Communicates in multiple languages to reach diverse populations.
- **Healthcare Awareness**: Provides information on preventive care, hygiene, nutrition, and common illnesses.
- **Symptom Guidance**: Uses rule-based and AI-driven flows to guide users toward self-care, doctor visits, or emergency care.
- **Scalable Infrastructure**: Cloud-ready deployment for large-scale usage.
- **Agentic AI Principles**: Designed with autonomy and adaptability for evolving healthcare needs.

---

## 🛠️ Tech Stack
- **Frameworks**: Rasa/Dialogflow for conversational AI
- **Languages**: TypeScript, JavaScript, CSS
- **APIs**: Government health APIs for verified information
- **Deployment**: Dockerized setup with cloud scalability

---

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+ recommended)
- Docker (optional, for containerized deployment)

### Installation
```bash
# Clone the repository
git clone https://github.com/mitarthpathak/SAGACITY.git

# Navigate into the project
cd SAGACITY

# Install dependencies
npm install

# Bash run
npm start

# Docker setup-
docker build -t sagacity-chatbot .
docker run -p 3000:3000 sagacity-chatbot
```
